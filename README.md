# Maximizing Revenue for Drivers Through Payment Type Hypothesis Testing

## Overview

This project focuses on analyzing the relationship between taxi fare amounts and customer payment methods using statistical analysis and hypothesis testing. The main objective is to determine whether payment type (Credit Card or Cash) has a significant impact on total fare revenue.

Using Python and data analysis techniques, the project extracts meaningful business insights that can help taxi drivers and companies maximize revenue and improve payment strategies.

---

# Objectives

* Analyze customer payment behavior.
* Compare fare amounts between different payment methods.
* Perform hypothesis testing to identify significant differences.
* Use statistical analysis to support business decision-making.
* Generate insights to help drivers maximize earnings.
